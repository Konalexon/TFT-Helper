## Installation

```bash
# Clone the repository
git clone https://github.com/Konalexon/TFT-Helper.git

# Navigate to electron folder
cd TFT-Helper/electron

# Install dependencies
npm install

# Run the app
npm start
```

## Build

```bash
# Build for Windows
npm run build:win
```

## Tech Stack

- **Electron** - Desktop app framework

- **MetaTFT Data** - Composition data source
